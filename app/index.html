初回設定

このツールはホーム画面に追加することをお勧めします。
ホーム画面に追加すると……

  • 開くのが楽
  • アプリのように使えるので、USEN推しリクなどが終わったら左上の✖で閉じれば続きが超簡単
  • とにかく楽ちん

以下にOS別の導入方法を記載しています。
もし上手くいかないとき、難しいときはお気軽にお問い合わせください。
mond　X

iPhone(iPad) / iOS の場合

  1. Safariでこのページを開く
  2. 「共有」ボタンを押す
    「」こんなボタン
  3. 「ホーム画面に追加」を選ぶ
    表示されないときは「もっとみる」をタップし、探してみてください
  4. 「WEBアプリとして開く」の横がのようになっていること（オン状態）を確認し、右上の「追加」を押す
  5. ホーム画面に「タムごとDaily」というアプリが追加されるので、それをタップして開いてください

Android

  1. Chromeでこのページを開く
  2. 右上のメニュー（）を押す
  3. 「インストール」または「ホーム画面に追加」を選ぶ
  4. 「追加」または「はい」を選ぶ
  5. ホーム画面に「タムごとDaily」というアプリが追加されるので、それをタップして開いてください

SpotifyやYouTubeがブラウザで開く場合は、各アプリを入れてログインしたうえで、Androidは対象アプリの「対応リンクを開く」をONにしてください。

使い方

基本の流れ

  1. SpotifyでBGMを選ぶ
    1日1曲聞くだけで月間リスナーに貢献♪
    最初に表示されたものの中に好みのものがなければ、「その他」をタップすると現在Spotifyで配信されている曲の一覧が表示されます。
    画面上部に「Spotifyで再生する」ボタンが表示されるので、それをタップしてください。
    曲が再生されたらアプリを切り替えて「タムごとDaily」に戻ってきてください。
    もし曲を再生したくないときは「BGMなし」を選ぶと次に進みます。
  2. 期間限定タスク
    〇日までなど、期間限定系のタスクが表示されます。
    終わってないもので今投票/リクエストなどしたいものがあったら✅をいれて「次へ」を押してください。
    • アプリ遷移するもの(SpotifyやYouTubeなど)の場合
      おそらく自動で遷移します。(しない場合は既定ブラウザで開くと遷移します)
      タスクが完了したらSpotify同様にアプリを切り替えて「タムごとDaily」に戻ってきてください。
    • アプリ遷移しないもの(アプリが切り替わらずWEBページが表示される)場合
      タスクが完了したら✖や完了などで閉じてください。
      XやLINEなどで開いたWEBページを閉じるときと同じ要領です。
    • 曲名を入力する必要があるもの
      ページを開くとき、各フォームに合わせた形でクリップボードにコピーされます。
      例：timeleszのRock this Partyをリクエストします！, timelesz/Rock this Party など
      もしフォームや入力方法の変更などがあった場合はご連絡ください。
    タスクが完了してアプリに戻ったりWEBページを閉じたりすると、「次へ」ボタンが表示されます。
    最後のタスクが完了すると次に移動します。
  3. デイリータスク
    毎日やれるタスクが表示されます。
    ページを開く前に注意事項ややり方が簡単に表示されるのでそれを参考にしてください。
    分かりにくいときはご連絡ください。
    • アプリ遷移するもの(SpotifyやYouTubeなど)の場合
      おそらく自動で遷移します。(しない場合は既定ブラウザで開くと遷移します)
      タスクが完了したらSpotify同様にアプリを切り替えて「タムごとDaily」に戻ってきてください。
    • アプリ遷移しないもの(アプリが切り替わらずWEBページが表示される)場合
      タスクが完了したら✖や完了などで閉じてください。
      XやLINEなどで開いたWEBページを閉じるときと同じ要領です。
    • 曲名を入力する必要があるもの
      ページを開くとき、各フォームに合わせた形でクリップボードにコピーされます。
      例：timeleszのRock this Partyをリクエストします！, timelesz/Rock this Party など
      もしフォームや入力方法の変更などがあった場合はご連絡ください。
    タスクが完了してアプリに戻ったりWEBページを閉じたりすると、「次へ」ボタンが表示されます。
    最後のタスクが完了すると、続けるかどうかの質問が出ます。(出ないときもあります)
    続けるときはまたタスクが表示されます。
    これまでと同じ要領で進めてください。
    続けないときは次に進みます。
  4. SNS共有
    こなしたタスクをSNSに共有できます。
    共有したいものを選択して希望のボタンを押してください。
    各SNSアプリで投稿が終わったらこのアプリに戻り、「次へ」ボタンを押してください。
  5. YouTube再生
    もし余力があったらYouTubeも再生していってください。
    再生リストや動画のの追加リクエストはmondまたはDMまたはリプライで受け付けています。

このツールについて
「タムごとDaily」は、timeleszを応援する毎日の行動をまとめて確認できる非公式ファンメイドツールです。
Spotifyを聴く、USEN推しリクをする、SNSでシェアする、期間限定のタスクを確認するなど、日々の推し活を少しでも忘れず・楽しく続けられるように作成しています。
掲載している情報やリンクはできるだけ確認していますが、手動管理のため内容が古くなる場合があります。
大事なお知らせや期限については、必ず公式情報もあわせてご確認ください。

このツールは個人による非公式ツールです。
所属事務所、各サービスとは関係ありません。

またこのツールで個人の特定につながる情報を収集することはありません。

ホーム画面に追加すると何が便利？

特に役立つのはUSENリクエスト・各種タスクなどのリンクを開いたあとです。
ホーム画面のアイコンから開いている場合、遷移先がアプリ内ブラウザのように表示されるため、 作業が終わったら画面を閉じるだけで、このツールに戻りやすくなります。

XやLINEの中でリンクを開いたときに、外部ページを閉じるだけで元の画面に戻れる感覚に近いです。
毎日のタスクを続けてこなすときに、戻る操作やページの探し直しが少なくなります。
SafariやChromeの普通のタブで開いたままだと、リンク先から戻るのが少し面倒になったり、
途中でページを見失いやすくなることがあります。

そのため、初回だけホーム画面に追加して、次回からは追加されたアイコンから開くのがおすすめです。
